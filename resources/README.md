The assets in this folder are the design source. PNG copies are rasterized
from them into the canonical locations used by Home Assistant and HACS
(`custom_components/southpool/icon.png`, `custom_components/southpool/brand/`,
`banner.png` in the repository root, etc.). To re-rasterize the PNGs from
the SVGs, an SVG-to-PNG tool is required (e.g. `cairosvg`, `rsvg-convert`,
Inkscape).
